This project allows users to draw in the air using hand gestures captured through a webcam.

## Features
- Air drawing using hand gestures
- Auto clear canvas gesture
- Eraser mode gesture
- Brush thickness control
- Color change gesture

## Technologies Used
- Python
- OpenCV
- Computer Vision

## How It Works
Hand gestures are detected using real-time video processing and mapped to drawing actions on a virtual canvas.

## Applications
- Digital art
- Online education
- Interactive presentations
- Assistive technology
